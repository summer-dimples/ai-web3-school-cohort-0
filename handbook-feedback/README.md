# Handbook Feedback

This directory contains feedback for the AI x Web3 School Handbook.

## Feedback Format

Each feedback item should include:

1. **Handbook Page URL** - Which page is the feedback about?
2. **Issue Description** - What's wrong or unclear?
3. **Suggested Fix** - How would you improve it?
4. **Source** - Your daily note or context where you found the issue

## Example

```markdown
### Feedback: RAG chapter missing vector DB comparison

- **Page**: https://aiweb3.school/zh/handbook/ai/rag/
- **Issue**: The RAG chapter mentions vector databases but doesn't compare popular options (Pinecone, Weaviate, Chroma)
- **Suggestion**: Add a comparison table of common vector DBs with pros/cons
- **Source**: daily/2026-05-18.md
```

## Filing Feedback

1. Create a new .md file for each feedback topic
2. Use the format above
3. Commit and push to your public repo
4. Optionally submit to: https://github.com/lxdao-official/aiweb3school/issues
