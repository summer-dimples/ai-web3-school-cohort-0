## Week 1 Learning Goals

**核心目标：**
- 理解 LLM、prompt、workflow、agent、tool use、AI coding 的基本概念
- 理解账户、钱包、签名、交易、Gas、合约、测试网和区块浏览器如何构成一条链上操作链
- 完成至少一次 AI 工具实践、一次测试网交互，以及一个 AI × Web3 最小交叉实验
- 建立权限、安全、人工确认、日志、验证材料和失败恢复意识

**Week 1 Task Checklist:**

模块 A｜AI 基础
- [ ] 搭建 learning agent（Claude Code / Codex / Hermes）并跑通一次学习任务
- [ ] 创建并初始化个人 GitHub repo（本 repo）
- [ ] 用 agent 生成一个可交互学习产物（小页面 / CLI / 流程图 / quiz）

模块 B｜Web3 基础
- [ ] 创建测试钱包，理解地址 / 助记词 / 私钥
- [ ] 切换到测试网，领取测试币，发送一笔测试交易
- [ ] 在 Remix 部署最小智能合约，完成读 + 写操作

模块 C｜最小交叉实验
- [ ] AI 生成 → 人工复核 → 钱包确认 → 链上执行 → 区块浏览器验证，完成一条完整流程
- [ ] 整理概念说明：LLM / workflow / agent / 钱包 / 签名 / 交易 / 合约放到同一任务链里解释
