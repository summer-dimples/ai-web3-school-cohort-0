# Week 1 模块 C｜最小交叉实验：AI 输出到链上执行

## 目标

把 AI 与 Web3 放到同一条任务链上。重点不是做复杂项目，而是第一次看见：
**AI 输出 → 人工复核 → 钱包确认 → 链上执行 → 区块浏览器验证** 之间如何衔接。

## 选择一条实验路线
