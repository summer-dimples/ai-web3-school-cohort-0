### Key Learnings（今日核心收获）

**密码学与 AI 的类比：**
1. 私钥 ≈ API Key，但泄露后果远更严重：无法重置，资产永久丢失
2. 数字签名 ≈ 请求签名，区别是每次签名都在授权一个不可撤销的链上动作
3. 哈希的"雪崩效应"保证了区块链的不可篡改性（改一个字符就改变了所有后续区块）

**智能合约核心理解：**
4. 合约 = 链上的确定性程序，部署后代码不变、任何人可调用、无法被停止
5. View/Pure 函数免费（读取），普通函数需要 Gas + 签名（写入）
6. AI 生成的合约代码必须经过安全审查，不能直接部署主网

**Indexing：**
7. Events 需要合约主动 emit，否则 Etherscan 的 Events 标签为空
8. The Graph = 区块链的"数据库索引"，用 GraphQL 查询，解决链上数据查询性能问题

**Security 四大漏洞：**
9. 重入攻击：先转账再改状态；修复：先改状态再转账（CEI 模式）
10. Solidity 0.8+ 自动处理溢出，旧版本必须用 SafeMath
11. 访问控制：用 `require(msg.sender == owner)` 或 OpenZeppelin Ownable
12. Front-Running：MEV Bot 看到内存池中的交易后抢先执行
